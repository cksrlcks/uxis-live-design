# 게스트 이름

뷰어는 **로그인 없이** 볼 수 있습니다. 다만 커서·코멘트·채팅에 표시될 **이름**이 필요합니다.
cova는 이름을 미리 묻지 않고, 처음에는 임시 게스트 이름을 자동으로 부여합니다.

## 자동 게스트 이름

처음 접속하면 **"Guest 1234"** 처럼 임시 이름과 색이 자동으로 정해집니다.
이 상태로도 시안을 보고, 커서를 움직이고, 채팅을 보낼 수 있습니다.

## 이름 바꾸기

오른쪽 위 접속자 바에서 내 게스트 이름을 클릭하면 그 자리에서 바로 수정할 수 있습니다.

1. 접속자 바의 내 이름을 클릭합니다.
2. 새 이름을 입력하고 체크(저장) 버튼 또는 Enter로 저장합니다.
3. 바꾼 이름은 **이 브라우저에 저장**되어, 다음에 다시 접속해도 그대로 유지됩니다.

::: info 🖼 이미지 자리
접속자 바 — 게스트 이름 인라인 편집(입력창 + 저장 버튼)
:::

::: tip 로그인하면
[스튜디오 계정](/studio/)으로 로그인한 상태면 게스트 이름 대신 **계정 이름**이 표시되며,
이때는 이름을 바꿀 수 없습니다(계정 이름 고정). 로그아웃하면 다시 저장해 둔 게스트
이름으로 돌아옵니다.
:::
