# 프리뷰

시안을 보는 방식은 두 가지입니다. **풀화면(슬라이드)** 과 **캔버스(자유 이동·확대)**.
캔버스 뷰 하단의 컨트롤 바에서 **"풀화면" / "캔버스"** 버튼으로 전환합니다.

## 풀화면 (슬라이드)

한 번에 한 페이지씩, 실제 크기로 보여주는 발표용 보기입니다.

- **다음 페이지** — 화면을 **클릭**하거나 키보드 **→** 키.
- **이전 페이지** — 키보드 **←** 키.
- **페이지 위치** — 오른쪽 아래에 **"3/12"** 처럼 현재/전체가 표시됩니다.

::: info 🖼 이미지 자리
풀화면 슬라이드 — 시안 한 페이지 + 우하단 페이지 인디케이터("3/12")
:::

## 캔버스 (Figma 스타일)

모든 페이지를 한 화면에 펼쳐 자유롭게 이동·확대하며 보는 방식입니다. 실시간 회의
(커서·코멘트·그리기)도 캔버스에서 이뤄집니다.

- **이동(Pan)** — 빈 곳을 **드래그**. 또는 **스페이스바 + 드래그**, **마우스 휠 클릭 드래그**.
- **확대/축소(Zoom)** — **마우스 휠** 스크롤.
- **처음 보기** — 들어오면 모든 페이지가 한눈에 들어오도록 자동으로 맞춰집니다.

### 하단 컨트롤 바

화면 아래 가운데에 떠 있는 바에서 다음을 조작합니다.

| 버튼 | 하는 일 |
|------|---------|
| **목록** | 안(Variant) 목록으로 돌아가기 |
| 안 이름들 | 볼 안 선택 |
| **v1 · v2 …** | 버전 선택(버전이 2개 이상일 때) |
| **나란히** | 두 안을 좌우로 비교(안이 2개 이상일 때) |
| **풀화면** | 풀화면 슬라이드로 전환 |
| **캔버스** | 캔버스 보기(현재 보기) |

컨트롤 바는 접을 수 있습니다(**"컨트롤러 숨기기" / "컨트롤러 펼치기"**).

::: info 🖼 이미지 자리
캔버스 뷰 — 펼쳐진 페이지들 + 하단 컨트롤 바(목록/안/버전/나란히/풀화면/캔버스)
:::

### 나란히 보기

**"나란히"** 를 누르면 여러 안을 좌우로 나란히 두고 비교합니다. 각 열 위에 안 이름이
고정되어 표시됩니다.

::: info 🖼 이미지 자리
나란히 보기 — 두 안을 좌우 컬럼으로 비교(상단에 안 이름 고정)
:::

::: tip
풀화면·캔버스·안·버전·나란히 상태는 주소(URL)에 담깁니다. 그래서 지금 보고 있는 화면
그대로 링크를 복사해 공유할 수 있습니다.
:::
