# 시안 열기

**cova 뷰어**는 공개된 시안을 **로그인 없이** 열람하는 화면입니다. 편집자가 공유한
공개 링크(`/p/{공개ID}` 또는 `/p/{도메인}`)로 접속합니다.

> 데스크탑 환경 권장 — 시안은 1920×1080(FHD) 기준으로 제작되어, 정확한 확인을 위해
> 큰 화면에서 보는 것이 좋습니다.

## 접근 단계

링크로 들어가면 시안의 공개 설정에 따라 세 가지로 갈립니다.

| 상태 | 화면 |
|------|------|
| **공개** (비번 없음) | 바로 열람 |
| **공개 + 비밀번호** | 비밀번호 입력 후 열람 |
| **비공개** | *"비공개 시안 — 시안을 확인하려면 관리자 권한이 필요합니다."* |

비공개 시안은 편집자·관리자가 [스튜디오](/studio/)에서 로그인해야 볼 수 있습니다.

## 비밀번호로 열기

비밀번호가 걸린 시안은 다음 화면이 먼저 나옵니다.

1. *"비밀번호가 필요한 시안입니다."* 안내 아래 **"비밀번호"** 칸에 비밀번호를 입력합니다.
   (비밀번호는 보통 담당자가 구두/메신저로 전달합니다.)
2. 다음에 또 묻지 않게 하려면 **"하루동안 기억하기"** 를 체크합니다.
3. **"열기"** 를 누릅니다. 틀리면 *"비밀번호가 올바르지 않습니다."* 가 표시됩니다.

::: info 🖼 이미지 자리
비밀번호 게이트 화면 — "비밀번호" 입력, "하루동안 기억하기" 체크, "열기" 버튼
:::

## 시안 화면

열람이 시작되면 먼저 **안(A·B…) 목록**이 카드로 보입니다. 카드에는 안 이름과
**"N pages"**(페이지 수)가 표시됩니다. 카드를 누르면 그 안의 프리뷰로 들어갑니다.

::: info 🖼 이미지 자리
뷰어 첫 화면 — 안(Variant) 카드 목록(안 이름 + 페이지 수)
:::
