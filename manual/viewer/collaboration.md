# 실시간 회의

캔버스 뷰에서는 같은 시안을 보는 사람들과 **실시간으로 회의**할 수 있습니다.
접속자 커서, 핀 코멘트, 채팅, 그리고 화이트보드(그리기)를 지원합니다.

## 접속자 보기 (커서)

같은 시안을 열고 있는 사람들의 **마우스 커서**가 캔버스 위에 실시간으로 보입니다.
화면 오른쪽 위 접속자 바에는 현재 접속 인원이 표시됩니다.

- 혼자면 **"혼자 보는 중"**, 여러 명이면 **"N명 접속"**.
- 접속자들의 아바타가 함께 표시됩니다.

::: info 🖼 이미지 자리
캔버스 위 다른 접속자 커서 + 우상단 접속자 바("N명 접속")
:::

## 핀 코멘트

시안의 특정 위치를 콕 집어 의견을 남깁니다.

1. 캔버스 하단의 모드 버튼에서 **"코멘트"** 를 선택합니다(커서가 십자 모양으로 바뀜).
2. 코멘트를 달 위치를 클릭합니다.
3. **"코멘트를 남겨주세요…"** 칸에 내용을 적고 **"저장"** 합니다(취소하려면 **"취소"** 또는 Esc).

> **로그인 필요** — 핀 코멘트는 로그인한 사용자만 남길 수 있습니다. 비로그인 상태로 핀을
> 찍으려 하면 *"로그인이 필요합니다 — 코멘트 핀을 남기려면 로그인해 주세요."* 안내가 뜨고,
> **"로그인"** 후 보던 화면으로 돌아옵니다.

찍힌 핀은 작성자 색의 말풍선으로 표시됩니다. 핀을 누르면:

- 작성자·시간과 내용이 보입니다.
- 내 코멘트는 **"수정" / "삭제"** 할 수 있습니다.
- 체크 아이콘으로 **해결됨** 처리할 수 있습니다(해결된 핀은 흐리게 + 초록 체크 표시).

### 코멘트 목록

하단 컨트롤의 **"목록"** 을 누르면 왼쪽에 코멘트 패널이 열립니다.
**"전체"** / **"미해결"** 탭으로 필터하며, 항목을 누르면 해당 핀으로 이동합니다.
코멘트가 없으면 *"아직 코멘트가 없습니다 — 시안 위를 클릭해 코멘트를 남겨보세요."*.

::: info 🖼 이미지 자리
핀 코멘트 — 캔버스 위 말풍선 핀 + 왼쪽 코멘트 목록 패널(전체/미해결 탭)
:::

## 채팅

접속자 바의 채팅 버튼(말풍선 아이콘)으로 **채팅 패널**을 엽니다.

1. 아래 입력창(**"메시지 입력…"**)에 내용을 적고 전송 버튼(또는 Enter)으로 보냅니다.
2. 내 메시지는 오른쪽, 다른 사람 메시지는 왼쪽에 표시됩니다.
3. 내가 보낸 메시지는 마우스를 올려 **수정/삭제**할 수 있습니다(수정 중 *"Enter 저장 · Esc 취소"*).

채팅 패널이 닫혀 있을 때 새 메시지가 오면 버튼에 안 읽은 개수가 배지로 표시됩니다.
메시지가 없으면 *"아직 메시지가 없습니다 — 첫 메시지를 남겨보세요."*.

::: info 🖼 이미지 자리
채팅 패널 — 좌우로 구분된 메시지 말풍선 + "메시지 입력…" 입력창
:::

## 화이트보드 (그리기)

편집자가 해당 시안의 [화이트보드 설정](/studio/proposal-detail)을 켜 둔 경우,
캔버스에서 시안 위에 자유롭게 그릴 수 있습니다(로그인 사용자).

- 하단 모드에서 **"그리기"**(펜)를 선택하면 그리기 모드가 됩니다.
- **"화이트보드"**(눈 아이콘)로 그린 내용을 표시/숨김 전환합니다.
- 펜 색상(6색)과 굵기(S·M·L), 지우개를 사용할 수 있습니다.

::: info 🖼 이미지 자리
화이트보드 — 시안 위에 그린 펜 스트로크 + 색상/굵기 도구
:::
