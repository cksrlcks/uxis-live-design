# 태그 설정

**태그 설정**(`/studio/tags`)에서는 시안에 붙일 **태그 구조**를 관리합니다. 여기서 만든
분류와 항목을 [시안 상세 → 태그관리](/studio/proposal-detail#태그관리)에서 각 시안에 부여합니다.

::: warning 관리자 전용
이 메뉴는 **관리자(admin)** 에게만 보입니다. 편집자는 접근할 수 없습니다.
:::

## 태그 구조: 구분과 항목

태그는 **구분(분류) → 항목** 의 2단 위계입니다.

- **구분(Group)** — 태그의 큰 분류(예: *목적*). 고정키인 **코드**(예: `purpose`)와 이름을 가집니다.
- **항목(Option)** — 구분 안의 선택지(예: *제안*). 마찬가지로 코드(예: `proposal`)와 이름을 가집니다.

> **코드(고정키)** 는 시스템이 태그를 식별하는 값으로, 추가할 때 한 번 정하면 바꾸지 않습니다.
> 이름·설명은 나중에 수정할 수 있습니다.

## 구분 관리

화면 상단에 제목 **"태그 설정"**, 설명 *"시안에 태깅할 구분과 항목을 관리합니다."*,
오른쪽에 **"구분 추가"** 버튼이 있습니다. 구분이 하나도 없으면
*"아직 구분이 없습니다. '구분 추가'로 시작하세요."* 가 표시됩니다.

1. **구분 추가** — **"구분 추가"** → "코드(고정키)"(예: `purpose`), "이름"(예: *목적*), "설명(선택)"
   입력 후 **"저장"**.
2. **구분 수정** — 카드의 연필 아이콘. 이름·설명만 바꿉니다(코드는 고정).
3. **구분 삭제** — 휴지통 아이콘. *"…구분을 삭제할까요?"* — 이 구분의 모든 항목과,
   항목이 선택된 시안의 태그 기록도 함께 삭제됩니다.

::: info 🖼 이미지 자리
태그 설정 — "구분 추가" 버튼과 구분 카드 목록(항목 수 배지·코드 표시)
:::

## 항목 관리

각 구분 카드 안에서 항목을 관리합니다.

1. **항목 추가** — 카드 아래 **"항목 추가"** → "코드(고정키)"(예: `proposal`), "이름"(예: *제안*),
   "설명(선택)" 입력 후 **"저장"**.
2. **항목 수정 / 삭제** — 항목 행에 마우스를 올리면 연필·휴지통 아이콘이 나타납니다.

항목이 없는 구분에는 *"항목이 없습니다."* 가 표시됩니다.

::: tip 코드 중복 주의
이미 쓰는 코드로 추가하면 *"추가에 실패했습니다(코드 중복일 수 있어요)"* 가 뜹니다.
구분·항목의 코드는 서로 겹치지 않게 정하세요.
:::
