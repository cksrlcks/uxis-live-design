# 시안 상세·버전

시안 하나를 열면(목록에서 **"수정하기"**) 상세 화면이 나옵니다. 상단에 시안 제목과 공개 URL이
표시되고, 본문은 **세 개의 탭**으로 나뉩니다.

| 탭 | 하는 일 |
|------|---------|
| **사이트설정** | 제목·참여자·공개 상태·비밀번호·도메인 등 시안 전체 설정 |
| **시안설정** | 안(A·B…)과 버전(v1·v2…) 관리, 이미지 미리보기 |
| **태그관리** | 시안에 태그 부여 |

> 편집자·관리자만 접근할 수 있습니다.

## 시안설정 — 안(Variant)과 버전

cova의 시안은 **안 → 버전 → 페이지(이미지)** 구조입니다.

- **안(Variant)** — 하나의 시안 안에서 비교하는 시안들(예: A안, B안).
- **버전(Version)** — 각 안의 히스토리(v1, v2, v3…). 새로 올릴 때마다 쌓입니다.
- **페이지** — 한 버전에 들어가는 이미지들.

### 안 관리

왼쪽 카드에서 안 목록을 관리합니다.

1. **안 추가** — 점선 **"안 추가"** 버튼으로 새 안(B, C…)을 만듭니다.
2. **이름 변경** — 안 옆 연필 아이콘 → 이름을 고치고 저장.
3. **순서 변경** — 그립 핸들을 끌어서 이동하거나 화살표 키(↑/↓)로 정렬.
4. **삭제** — 휴지통 아이콘. *"…안을 삭제할까요? 이 안의 모든 버전이 삭제됩니다."* 확인.
   (안은 최소 1개가 있어야 하므로 마지막 안은 삭제할 수 없습니다.)

### 버전 관리와 이미지

오른쪽 미리보기에서 선택한 안의 버전을 관리합니다.

1. 버전 세그먼트(**v1 · v2 …**)에서 볼 버전을 고릅니다.
2. **"새 버전"** 으로 빈 버전을 추가한 뒤, 이미지 그리드에서 이미지를 올립니다.
3. 이전 버전을 보는 중이면 **"복원"** 버튼으로 그 버전을 현재 버전으로 되돌립니다.
   (현재 버전에는 **"현재 버전"** 으로만 표시됩니다.)
4. 버전 삭제는 휴지통 아이콘. *"v… 버전을 삭제할까요?"* 확인. (버전도 최소 1개 필요)

::: info 🖼 이미지 자리
시안설정 탭 — 왼쪽 안 목록("안 추가") + 오른쪽 버전 세그먼트(v1·v2)와 이미지 그리드
:::

## 사이트설정 — 공개·비밀번호·도메인

각 항목은 카드로 나뉘며, 바꾼 뒤 **"저장"** 하면 *"저장했습니다"* 로 즉시 반영됩니다.

- **제목** — 시안 목록과 뷰어에 표시되는 이름.
- **참여자** — 쉼표로 여러 명 입력(예: *"홍길동, 김철수"*). 목록 표시·검색에 사용됩니다.
- **공개 상태** — **공개 / 비공개** 토글. *"비공개는 관리자만, 공개는 링크를 아는 누구나 볼 수 있습니다."*
- **접근 비밀번호** — 공개 시안에만 적용. "4자 이상"으로 설정/변경하거나 **"비번 해제"**.
  설정하면 뷰어에서 열람 시 비밀번호를 묻습니다. → [뷰어: 시안 열기](/viewer/)
- **공개 도메인** — 공개 URL에 쓰는 식별자(소문자·숫자·하이픈). **"중복확인"** 후 **"설정/변경"**.
  설정하면 `/p/{도메인}` 으로도 접속됩니다.
- **화이트보드** — 켜면 뷰어 캔버스에서 로그인 사용자가 시안 위에 그릴 수 있습니다.
  → [뷰어: 실시간 회의](/viewer/collaboration)
- **유시스웍스 노출** — 켜면 유시스웍스(포트폴리오/갤러리)에 노출합니다(공개 링크 접근과는 별개).
- **피그마 링크** — 원본 Figma 파일 주소를 연결(`figma.com` 주소만).

::: info 🖼 이미지 자리
사이트설정 탭 — 공개 상태/접근 비밀번호/공개 도메인 카드
:::

### 시안 삭제

맨 아래 위험 구역의 **"시안 삭제"** 는 *"모든 버전과 이미지가 영구 삭제됩니다. 되돌릴 수 없습니다."*
확인 후 진행됩니다. 신중히 사용하세요.

## 태그관리

**태그관리** 탭에서 [태그 설정](/studio/tags)에 등록된 분류별로 항목을 골라 시안에 부여합니다.
고른 뒤 **"저장"** 하면 *"태그를 저장했습니다"* 로 반영됩니다. 분류가 비어 있으면
*"등록된 태그 분류가 없습니다. 관리자에게 문의하세요."* 가 표시됩니다.

::: info 🖼 이미지 자리
태그관리 탭 — 분류별 태그 항목 선택 + "저장" 버튼
:::
