# 시안

**시안**은 cova의 기본 단위입니다. 디자인 이미지를 올려 버전으로 관리하고, 공개 링크로
공유해 실시간으로 회의합니다. 이 화면(`/studio/proposals`)에서는 **시안 목록**을 보고
**새 시안**을 만듭니다.

> 편집자·관리자라면 누구나 **모든 시안**을 보고 만들 수 있습니다(전체 편집 권한).

## 시안 목록 보기

상단에 제목 **"시안"** 과 오른쪽에 **"새 시안"** 버튼이 있습니다. 그 아래 검색창과 표가 있습니다.

- **검색** — *"제목·참여자·도메인 검색"* 으로 빠르게 찾습니다. 표 위에 **"전체 N개"** 로 개수가 보입니다.
- **표 컬럼** — 제목 · 참여자 · 공개 ID · 공개 도메인 · 상태 · 태깅 · 작성일 · 최근수정일 · 작업.
- **상태 배지** — 시안의 공개 상태를 한눈에 보여줍니다.

| 배지 | 의미 |
|------|------|
| **공개** | 링크를 아는 누구나 열람 가능 |
| **비공개** | 관리자(편집자·관리자)만 열람 가능 |
| **비번** | 공개 시안에 접근 비밀번호가 걸림 |
| **노출** | 유시스웍스(포트폴리오/갤러리)에 노출 중 |

::: info 🖼 이미지 자리
시안 목록 — 상단 "시안" 제목 + "새 시안" 버튼, 검색창, 표(상태 배지 포함)
:::

### 작업 메뉴 (행 오른쪽 "작업")

각 행의 **"작업"** 메뉴에서 다음을 할 수 있습니다.

- **수정하기** — [시안 상세](/studio/proposal-detail) 편집 화면으로 이동.
- **공개ID 링크 복사 / 열기** — `/p/{공개ID}` 형태의 공개 링크.
- **도메인 링크 복사 / 열기** — 공개 도메인을 설정한 경우 `/p/{도메인}` 링크.

링크를 복사하면 *"시안 링크를 복사했습니다"* 안내가 뜹니다.

## 새 시안 만들기

방법은 두 가지입니다.

### 빠르게 만들기 (다이얼로그)

목록에서 **"새 시안"** 을 누르면 작은 창이 뜹니다.

1. **"시안 이름"** 에 제목을 입력합니다(예: *"메인 페이지 리뉴얼"*).
2. **"만들기"** 를 누릅니다.
3. *"시안을 만들었습니다"* 안내와 함께 빈 시안이 생성됩니다.

### 전체 폼으로 만들기

`/studio/proposals/new` 에서는 안내 문구 *"제목과 이미지를 올리면 v1이 자동 생성됩니다."*
와 함께 만듭니다.

1. **"제목"** 을 입력합니다.
2. **"시안 만들기"** 를 누릅니다. 제목만 입력하면 빈 **v1**(버전 1)이 자동으로 만들어집니다.
3. 이미지는 만든 뒤 [시안 상세](/studio/proposal-detail)에서 추가·교체합니다.

::: info 🖼 이미지 자리
"새 시안 만들기" 다이얼로그 — "시안 이름" 입력과 "만들기" 버튼
:::

::: tip
시안을 만든 직후에는 이미지가 없는 상태입니다. 이어서 [시안 상세](/studio/proposal-detail)에서
안(A·B)과 버전을 구성하고 이미지를 올리세요.
:::
