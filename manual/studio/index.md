# 시작하기

**cova 스튜디오**는 편집자·관리자가 시안을 만들고, 버전과 공개 설정을 관리하는 작업 공간입니다.
주소는 `/studio` 이며, 로그인한 **편집자(editor)** 또는 **관리자(admin)** 만 들어올 수 있습니다.

## 가입부터 권한 받기까지

cova는 가입 직후에는 아무 권한이 없는 **승인 대기(pending)** 상태로 시작합니다.
관리자가 승인해 **편집자**나 **관리자** 권한을 주면 그때부터 시안을 관리할 수 있습니다.

1. **회원가입** — `/signup` 에서 "이름 · 이메일 · 비밀번호"를 입력하고 **"가입하기"** 를 누릅니다.
   - 비밀번호는 "영문·숫자·특수문자 포함 8자 이상" 이어야 합니다.
2. **승인 대기** — 가입하면 **"승인 대기 중"** 화면으로 이동합니다.
   - 안내: *"관리자 승인 후 시안을 관리할 수 있습니다. 승인이 완료되면 바로 이용하실 수 있어요."*
3. **관리자 승인** — 관리자가 [사용자 관리](/studio/users)에서 역할을 **편집자** 또는 **관리자** 로 바꿔줍니다.
4. **이용 시작** — 승인 후 다시 로그인하면 스튜디오에 진입할 수 있습니다.

::: info 🖼 이미지 자리
회원가입 화면 — "이름 / 이메일 / 비밀번호" 입력 폼과 "가입하기" 버튼
:::

::: info 🖼 이미지 자리
"승인 대기 중" 화면 — 시계 아이콘 + 안내 문구
:::

::: tip 첫 관리자(부트스트랩)
모든 계정은 `pending` 으로 시작하므로, 맨 처음 관리자는 운영자가 데이터베이스(Supabase)에서
직접 `admin` 으로 지정합니다. 이후부터는 관리자가 앱 안에서 다른 사용자를 승인·승격할 수 있습니다.
:::

## 로그인

`/login` 에서 로그인합니다. 두 가지 방법이 있습니다.

- **이메일 로그인** — "이메일 · 비밀번호" 입력 후 **"로그인"**.
- **구글 로그인** — **"Google로 계속하기"** 버튼으로 구글 계정 로그인.

::: info 🖼 이미지 자리
로그인 화면 — 이메일/비밀번호 입력, "로그인", "또는", "Google로 계속하기" 버튼
:::

### 비밀번호를 잊었을 때

로그인 화면의 **"비밀번호를 잊으셨나요?"** 링크 → `/forgot-password` 에서 가입한 이메일을
입력하면 재설정 메일이 발송됩니다. 메일의 링크로 새 비밀번호를 설정한 뒤 다시 로그인합니다.

## 스튜디오 화면 구성

진입하면 왼쪽에 사이드바가 있습니다. 권한에 따라 메뉴가 다르게 보입니다.

| 메뉴 | 경로 | 권한 |
|------|------|------|
| **시안** | `/studio/proposals` | 편집자·관리자 |
| **태그 설정** | `/studio/tags` | 관리자 전용 |
| **사용자 관리** | `/studio/users` | 관리자 전용 |
| **홈으로** | `/` | 모두 |

사이드바 맨 아래에는 내 이름·이메일이 표시되며, 클릭하면 [내 계정](/studio/account)으로,
로그아웃 아이콘으로 로그아웃합니다.

::: info 🖼 이미지 자리
스튜디오 첫 진입 화면 — 왼쪽 사이드바(시안/태그 설정/사용자 관리)와 본문 영역
:::
