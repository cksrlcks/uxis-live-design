# 내 계정

**내 계정**(`/me`)에서는 내 프로필 정보를 확인하고, **표시 이름**과 **비밀번호**를 바꿉니다.
사이드바 아래의 내 이름·이메일 영역을 클릭하면 이동합니다.

> 로그인한 사용자만 접근할 수 있습니다(로그아웃 상태면 로그인 화면으로 이동).

## 화면 구성

- **아바타** — 표시 이름(또는 이메일)의 첫 글자.
- **표시 이름** — 옆의 변경 버튼으로 수정합니다. 시안·뷰어·채팅 등에 표시되는 이름입니다.
- **이메일** — 가입한 이메일(변경 불가).
- **가입일** — 예: *"가입일 2026년 6월 24일"*.

::: info 🖼 이미지 자리
내 계정 화면 — 아바타, 표시 이름(변경 버튼), 이메일, 가입일
:::

## 표시 이름 바꾸기

표시 이름 옆 변경 버튼을 누르면 다이얼로그가 열립니다. 새 이름을 입력하고 저장하면
바로 반영됩니다.

## 비밀번호 바꾸기

비밀번호 변경 다이얼로그에서 새 비밀번호를 설정합니다.
(구글 로그인만 사용하는 계정이라면 비밀번호 변경이 필요 없을 수 있습니다.)

화면 아래의 **"홈으로 돌아가기"** 링크로 처음 화면으로 나갑니다.
