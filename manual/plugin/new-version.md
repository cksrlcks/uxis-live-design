# 새 버전·새 안 올리기

이미 만든 시안에 **새 버전**을 올리거나 **새 안(B·C…)** 을 추가하고, 개별 페이지를 **교체**합니다.
모두 **"기존 시안"** 탭에서 합니다.

## 시안 찾기

1. **"기존 시안"** 탭을 엽니다. 내 시안 목록이 보입니다.
2. 위 검색창으로 제목을 검색해 시안을 찾습니다.
3. 시안을 누르면 그 시안의 **안 목록**이 열립니다.

::: info 🖼 이미지 자리
"기존 시안" 탭 — 시안 검색창과 시안 목록
:::

## 안 목록에서 — 새 버전 / 새 안

안 목록에서는 각 안이 **"안 A"**, **"안 B"** … 로 보이고, 그 아래 *"N개 버전 · 현재 M장"* 이
표시됩니다. (먼저 피그마에서 올릴 프레임을 선택해 두어야 버튼이 활성화됩니다.)

- **새 버전 올리기** — 안 오른쪽의 **"새 버전"** 버튼. 선택한 프레임이 그 안의 새 버전(v2, v3…)
  으로 올라갑니다. 끝나면 *"안 A v2 업로드됨"* 알림.
- **새 안 추가** — 아래 **"＋ 새 안 추가 (선택 프레임)"** 버튼. 선택한 프레임으로 새 안(B, C…)을
  만듭니다(이름은 자동 부여). 끝나면 *"새 안 B 추가됨"* 알림.
- **← 목록** 으로 시안 목록으로 돌아갑니다.

::: info 🖼 이미지 자리
안 목록 — 각 안의 "새 버전" 버튼과 하단 "＋ 새 안 추가 (선택 프레임)"
:::

## 페이지 교체

안을 누르면 그 안의 **현재 버전 페이지 목록**이 열립니다(제목: *"안 A · 현재 버전"*).
각 페이지는 *"1페이지"*, *"2페이지"* … 와 크기가 표시됩니다.

1. 피그마에서 교체할 새 프레임 **하나**를 선택합니다.
   (안내: *"교체는 캔버스에서 선택한 첫 프레임을 사용합니다."*)
2. 바꾸려는 페이지의 **"교체"** 버튼을 누릅니다.
3. 끝나면 *"3페이지 교체됨"* 처럼 알림이 뜹니다.
- **← 안 목록** 으로 돌아갑니다.

::: info 🖼 이미지 자리
페이지 목록 — 각 페이지 썸네일·크기와 "교체" 버튼
:::
