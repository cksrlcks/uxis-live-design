# 새 시안 만들기

피그마에서 선택한 프레임들을 그대로 올려 **새 시안**을 만듭니다.

## 준비: 프레임 선택

먼저 피그마 캔버스에서 시안으로 만들 **프레임을 선택**합니다.

- 여러 개를 선택하면 **선택한 순서대로 페이지**가 됩니다.
- 선택하면 플러그인 상단에 **"선택된 프레임: N개"** 로 표시됩니다.
  (아무것도 선택하지 않으면 *"Figma에서 올릴 프레임을 선택하세요"*.)

::: tip 이미지 형식
프레임은 1배율 PNG로 내보내집니다. 아주 큰 프레임은 용량 제한(약 25MB)에 걸릴 수 있으니
지나치게 큰 단일 프레임은 나눠서 올리세요.
:::

## 만들기

1. **"새 시안"** 탭을 엽니다.
2. **"제목"** 칸에 시안 이름을 입력합니다(플레이스홀더: *"시안 제목"*).
3. **"선택한 프레임으로 새 시안 만들기"** 를 누릅니다.
   (제목이 비어 있거나 선택한 프레임이 없으면 버튼이 비활성화됩니다.)
4. 업로드가 진행되며 상태줄에 *"업로드 중 3/8…"* 처럼 진행 상황이 표시됩니다.
5. 끝나면 *"새 시안 '제목' 생성됨"* 알림이 뜨고, 제목 칸이 비워집니다.

::: info 🖼 이미지 자리
"새 시안" 탭 — "제목" 입력과 "선택한 프레임으로 새 시안 만들기" 버튼
:::

## 만든 뒤 — 관리화면 열기

업로드가 끝나면 하단에 **"관리화면 열기 ↗"** 버튼이 나타납니다. 누르면 브라우저에서
방금 만든 시안의 [스튜디오 상세 화면](/studio/proposal-detail)(`/studio/proposals/{id}`)이 열립니다.
거기서 공개 설정·버전·태그를 마저 정리하세요.

::: tip
새로 만든 시안은 기본적으로 **안 A · v1** 구조로 시작합니다. 다른 안이나 다음 버전을 추가하려면
[새 버전·새 안 올리기](/plugin/new-version)를 참고하세요.
:::
