# 소개·설치

**cova Figma 플러그인**은 피그마에서 선택한 **프레임을 그대로 cova 시안으로 올리는** 도구입니다.
피그마를 떠나지 않고 새 시안을 만들거나, 기존 시안에 새 버전·새 안을 추가할 수 있습니다.

플러그인으로 할 수 있는 일:

- cova 계정으로 **로그인**(브라우저 연동)
- 선택한 프레임으로 **새 시안 만들기**
- 기존 시안에 **새 버전 / 새 안(A·B…) 올리기**
- 특정 페이지 이미지 **교체**

> 업로드는 **편집자·관리자** 계정만 할 수 있습니다. 권한이 없는 계정으로 로그인하면
> *"편집 권한이 없는 계정입니다. 업로드가 막힙니다."* 경고가 표시됩니다.
> 권한은 [스튜디오 사용자 관리](/studio/users)에서 부여합니다.

## 설치

플러그인은 피그마 **데스크톱 앱**에서 매니페스트로 불러옵니다.

1. 피그마 데스크톱 앱을 엽니다.
2. 메뉴 → **Plugins → Development → Import plugin from manifest…**
3. 플러그인 폴더의 **`manifest.json`** 을 선택합니다.
4. 이후 **Plugins → Development → cova** 로 실행합니다.

::: info 🖼 이미지 자리
피그마 메뉴 — Plugins → Development → Import plugin from manifest…
:::

::: tip 개발자용 빌드
소스에서 직접 빌드하는 경우 `figma-plugin/` 에서 `npm install` 후 `npm run build` 를 실행하면
`dist/ui.html` 과 `dist/main.js` 가 생성됩니다(매니페스트가 이 파일들을 가리킵니다).
:::

## 화면 한눈에 보기

플러그인을 실행하면 위에서부터 이렇게 구성됩니다.

- **로그인 영역** — 로그인 전에는 로그인 화면, 로그인 후에는 계정 정보와 로그아웃.
- **선택 표시줄** — *"Figma에서 올릴 프레임을 선택하세요"* / 선택하면 *"선택된 프레임: N개"*.
- **탭** — **"새 시안"** / **"기존 시안"**.
- **상태줄** — 업로드 진행 상황과, 완료 후 **"관리화면 열기 ↗"** 버튼.

::: info 🖼 이미지 자리
플러그인 첫 화면 — 선택 표시줄 + "새 시안"/"기존 시안" 탭
:::
