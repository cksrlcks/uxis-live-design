# 로그인

플러그인 로그인은 **브라우저 연동(페어링)** 방식입니다. 플러그인 안에서 직접 비밀번호를
입력하지 않고, 웹 브라우저에서 평소처럼 로그인하면 플러그인이 자동으로 연결됩니다.
(이미 cova 웹에 로그인돼 있으면 클릭 몇 번으로 끝납니다.)

## 로그인하기

1. 플러그인을 실행하면 로그인 화면이 보입니다.
   제목 **"로그인"**, 안내 *"브라우저에서 로그인하면 자동으로 연결됩니다"*.
2. **"로그인하기"** 버튼을 누릅니다. 기본 브라우저가 열립니다.
3. 플러그인은 *"로그인 중입니다… 열린 브라우저에서 로그인을 완료해주세요."* 상태로 기다립니다.
4. 브라우저에서 cova에 로그인합니다(이미 로그인돼 있으면 건너뜀).
   완료되면 *"로그인 완료 — 피그마로 돌아가면 자동으로 로그인됩니다. 이 창은 닫아도 됩니다."*
   가 표시됩니다.
5. 피그마로 돌아오면 플러그인이 자동으로 로그인 상태가 됩니다.

::: info 🖼 이미지 자리
플러그인 로그인 화면 — "로그인" 제목과 "로그인하기" 버튼
:::

::: info 🖼 이미지 자리
브라우저 로그인 완료 화면 — "로그인 완료 / 피그마로 돌아가면 자동으로 로그인됩니다."
:::

기다리는 중에 그만두려면 **"취소"** 를 누릅니다. 계정이 없다면 **"계정이 없으신가요? 회원가입"**
링크로 가입합니다(가입 후 [관리자 승인](/studio/users)을 받아야 업로드할 수 있습니다).

## 로그인 유지 / 로그아웃

- 한 번 로그인하면 토큰이 플러그인에 저장되어 **다음에 다시 열어도 로그인 상태가 유지**됩니다.
- 로그인 후 상단에 계정 정보가 보이며, 거기서 **로그아웃**할 수 있습니다.

::: warning 편집 권한 필요
편집자·관리자가 아닌 계정으로 로그인하면 *"편집 권한이 없는 계정입니다. 업로드가 막힙니다."*
경고가 뜨고 업로드가 되지 않습니다. [사용자 관리](/studio/users)에서 권한을 받으세요.
:::
